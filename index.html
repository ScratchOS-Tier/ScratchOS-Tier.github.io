Free UI

几乎全能的老牌 SCOS

点击查看详情

该作品的作者

shxUwU

优点

  • 双层图标，过渡流畅
  • 功能较多
  • 稳定性强，问题少
  • 代码开源

缺点

  • 部分中等性能设备运行会卡顿
  • 代码量庞大且对新手来说难以编辑
  • 使用了大量扩展，可能造成兼容性问题

总结

Free UI 目前可以说是最强的移动端 SCOS，动画无可挑剔，功能也比较多，除了性能和改编，几乎没有弱点。给个 S，留点进步空间。

Clear UI

轻量稳定，单层图标的扛把子

点击查看详情

该作品的作者

半生风血

优点

  • 轻量化设计，运行十分流畅
  • 没有使用任何扩展，原版 Scratch 也可运行
  • 代码开源且易读，结构清晰
  • 极致的稳定性，目前没有发现任何问题
  • 方便用户的细节和小彩蛋一堆
  • 作者主打一个听劝
  • 整个作品没有照搬任何其它作品的代码，完全自研
  • Clearly 动态壁纸和 Cleak 触控系统、完美跟随算法

缺点

  • 目前应用和功能较少
  • 使用单层图标动画而非双层

总结

Clear UI NEXT 注重于细节和流畅度，稳定性极高，体验者用连点器没有测出任何问题；文件大小仅有 333 KB。该作品是其作者开发 3 年 SCOS 的经验大成。

Fix UI 1
